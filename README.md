# AI Research Automation System

An AI-powered research automation platform built using Groq, Tavily, LangChain and Streamlit.

## Features

- Web search using Tavily API
- Intelligent URL extraction
- Web scraping and content retrieval
- AI-generated structured research reports
- Critic/reviewer pipeline for report evaluation
- Streamlit UI for interactive usage
- Multi-stage orchestration workflow

## Tech Stack

- Python
- LangChain
- Groq API
- Tavily Search API
- BeautifulSoup
- Streamlit

## Workflow

1. Search latest information
2. Extract relevant URLs
3. Scrape article content
4. Generate research report
5. Critique and evaluate report quality

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
